QiSession メソッド

QiSession 型で公開されるメンバーは以下のとおりです。

メソッド
  名前説明
Close
セッションを閉じます。
Connect
指定したアドレスへの接続を試みます。
Create
非接続状態で新しくセッションを生成します。
Create(String)
接続先アドレスを指定し、接続が完了した状態のセッションを取得します。
Destroy
セッションを破棄します。
Equals
指定した Object が、現在の Object と等しいかどうかを判断します。
(Object から継承されます。)
GetEndpoints
セッションのエンドポイントを取得します。
GetHashCode
特定の型のハッシュ関数として機能します。
(Object から継承されます。)
GetService
サービス名を指定してサービスを取得します。この関数はGetServiceAsync(String)の待機結果を返します。
GetServiceAsync
サービス名を指定してサービスを取得します。
GetServices
サービス一覧を文字列の配列として取得します。
GetServicesAsync
サービス名の一覧を取得します。
GetType
現在のインスタンスの Type を取得します。
(Object から継承されます。)
GetUrl
(動作未確認)セッションに対応したURLを取得します。
Listen
セッションをリスンします。
RegisterService
サービスに名前を付けて登録します。
SetIdentity
(動作未確認)セッションの一意識別子を設定します。
ToString
現在のオブジェクトを表す文字列を返します。
(Object から継承されます。)
UnregisterService
IDを指定してサービスを登録解除します。

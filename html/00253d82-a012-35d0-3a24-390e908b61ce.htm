QiMethod メソッド

QiMethod 型で公開されるメンバーは以下のとおりです。

メソッド
  名前説明
Call
関数を同期的に呼び出し、結果を取得します。
CallObject
関数を同期的に呼び出し、結果となるQiObject型変数を取得します。
Equals
指定した Object が、現在の Object と等しいかどうかを判断します。
(Object から継承されます。)
GetHashCode
特定の型のハッシュ関数として機能します。
(Object から継承されます。)
GetType
現在のインスタンスの Type を取得します。
(Object から継承されます。)
Post
関数を非同期で呼び出し、進行状態の確認に利用可能なIDを発行します。
ToString
現在のオブジェクトを表す文字列を返します。
(Object から継承されます。)

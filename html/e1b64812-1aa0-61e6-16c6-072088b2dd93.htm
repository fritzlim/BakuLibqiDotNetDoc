QiMethodInfo プロパティ

QiMethodInfo 型で公開されるメンバーは以下のとおりです。

プロパティ
  名前説明
ArgumentSignature
引数の型に対応するシグネチャ文字列を取得します。引数はタプルで一括りになっています。
ArgumentsInfo
引数のインフォメーション一覧(ロジック的には割とどうでもいい)
Description
メソッドの役割についての要約文を取得します。
Name
メソッド名を取得します。
ReturnValueDescription
戻り値の意味についての要約文を取得します。
ReturnValueSignature
戻り値の型に対応するシグネチャ文字列を取得します。
UID
メソッドに割り当てられたIDを取得します。

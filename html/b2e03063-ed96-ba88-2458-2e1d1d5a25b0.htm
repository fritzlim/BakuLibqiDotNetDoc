QiObject メソッド

QiObject 型で公開されるメンバーは以下のとおりです。

メソッド
  名前説明
CallDirect
自力でシグネチャを正しく定義してタプルを渡し、関数を呼び出します。 ラッパーの動作不良に備えて公開されており、通常はItemStringで選択したメソッドを用いてください。
ConnectSignal
シグナル(イベント)にハンドラを登録します。
Destroy
インスタンスを破棄します。
DisconnectSignal(UInt64)
識別IDを師弟してシグナルの登録を解除します。
DisconnectSignal(ActionQiValue)
ハンドラーを指定してシグナルへの登録を解除します。
Equals
指定した Object が、現在の Object と等しいかどうかを判断します。
(Object から継承されます。)
GetHashCode
特定の型のハッシュ関数として機能します。
(Object から継承されます。)
GetMetaObject
サービスの内部情報を表すメタオブジェクトを取得します。
GetProperty
(動作未確認)オブジェクトのプロパティ値を取得します。
GetType
現在のインスタンスの Type を取得します。
(Object から継承されます。)
PostDirect
自力でシグネチャを正しく定義してタプルを渡し、関数を非同期で呼び出します。 デバッグ目的で公開されており、普通はItemStringで選択したQiMethodで Post(QiAnyValue)を使用してください。
SetProperty
(動作未確認)オブジェクトのプロパティに値を設定します。
ToString
現在のオブジェクトを表す文字列を返します。
(Object から継承されます。)
